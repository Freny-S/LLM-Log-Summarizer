# LLM-Log-Summarizer

A working AI-powered log summarizer that takes raw application logs and returns structured, human-readable analysis. It includes attributes like severity, root cause, affected services, and whether the issue resolved itself.

A REST API that wraps the summarizer, so instead of running a script manually, any app or service can now send logs to the endpoint and get back structured analysis.

A clean two-tab web interface that any non-technical person can use. No API knowledge needed, no terminal, just paste or upload and get results.
